# 🔐 Password Validation System (Java)

## 📌 Overview

A Java Swing-based application that validates passwords using rules like length, character types, and balanced brackets. It uses OOP concepts, stack, and exception handling.

## 🎯 Features

* Checks uppercase, lowercase, digit, special character
* Validates balanced brackets using stack
* Displays error messages with custom exceptions
* Simple GUI using Swing

## ▶️ Run

```bash
javac MainApp.java
java MainApp
```
